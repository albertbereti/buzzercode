# 📖 Buzzercode Operator Handbook

**CEO Guide to 24/7 Autonomous scaling**

## 🏗️ Architecture Overview

Buzzercode is a tiered organism designed for zero-human maintenance:

- **Tier 1 (Guardian)**: Maintains the heartbeat. Cleans up stale data. Drives SMS conversion.
- **Tier 2 (Overmind)**: Evolves the economy. Adjusts pricing. Generates "Synthetic Interest" to keep creators incentivized.
- **Tier 3 (Marketing)**: Programmatically renders Remotion video ads and prepares them for social APIs.

## 💰 Profit Levers (How to make money)

1. **The "Priority Hook"**: In `revenue_engine.js`, users pay $1 to jump the queue. This is pure arbitrage.
2. **Subscriptions (Buzz Gold)**: Fixed monthly revenue in exchange for creator perks.
3. **The Darwin Effect**: Automated supply/demand pricing ensures the marketplace never stalls.

## 📉 Scaling Protocol

If call volume exceeds 1,000/day:

1. Increase `priorityFee` in `revenue_engine.js` to $5.00.
2. Activate "VIP Only" mode in `index.js`.
3. Deploy the system to GCP Cloud Run for infinite horizontal scaling.
